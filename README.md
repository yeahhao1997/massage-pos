# 按摩院柜台系统（Member POS）

参考 Fresha 的单店按摩院柜台系统，核心是**会员卡**，含预约排班、收银结账、报表提成。

## 怎么用

**日常使用**：双击 `启动系统.bat`，等窗口显示地址后：
- 柜台电脑浏览器打开 `http://localhost:3000`
- 手机/平板连**同一个 WiFi**，打开窗口里「局域网」那一行地址（如 `http://192.168.x.x:3000`）

关闭：关掉那个黑窗口即可。数据全部保存在 `data/pos.db` 一个文件里，**拷走它就是完整备份**。

> 首次启动会自动安装依赖、打包前端，稍等 1~2 分钟。之后启动很快。

**改代码调试**：双击 `开发模式.bat`，前端 `http://localhost:5173` 热更新。

## 功能模块

| 模块 | 说明 |
|---|---|
| 💳 会员 | 开卡、储值充值（含赠送/积分）、余额扣款、次卡核销、消费流水 |
| 📅 预约 | 按日排班、技师时间冲突检测、到店/完成/取消状态流转 |
| 🧾 收银 | 服务+商品开单、按技师记提成、余额/微信/现金等多种支付 |
| 📊 报表 | 营业额、支付方式分布、技师业绩提成、近30天趋势 |
| ⚙️ 设置 | 员工技师、服务项目、商品、房间、卡类型维护 |
| 🏠 工作台 | 今日营业额/充值/新会员/预约一览 |

## 技术栈

- **数据库**：SQLite（Node 24 内置 `node:sqlite`，零依赖免编译），金额按「分」存整数
- **后端**：Node.js + Express，REST API 在 `server/routes/`
- **前端**：React + Vite + Tailwind，自适应桌面和手机

## 目录

```
server/
  db.js           数据库结构 + 演示数据
  index.js        服务入口
  routes/         members 会员 / catalog 资料 / appointments 预约 / orders 订单 / reports 报表
src/
  pages/          6 个页面
  App.jsx         布局 + 路由
data/pos.db       数据文件（备份就拷它）
```
